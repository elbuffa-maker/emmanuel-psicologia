# Emmanuel Martinez Buffa - Sitio Web

Sitio web profesional para psicólogo especializado en desarrollo personal.

## Características

- Diseño minimalista con paleta rosa pastel
- Sistema de reservas para sesiones de terapia
- Sistema de solicitud para publicidad y colaboraciones
- Totalmente responsivo
- Optimizado para Netlify

## Estructura

- `index.html` - Página principal
- `styles.css` - Estilos
- `script.js` - Funcionalidad
- `netlify.toml` - Configuración de Netlify

## Despliegue en Netlify

1. Sube el contenido de esta carpeta a Netlify
2. Netlify detectará automáticamente `netlify.toml`
3. El sitio estará en vivo en segundos
